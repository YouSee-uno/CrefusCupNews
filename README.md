# CrefusCup 順位速報

CrefusCup の順位速報を表示するシンプルな静的ページです。Google スプレッドシートの公開データを読み込み、コース別に順位表を表示します。

## 概要

- コース別タブ切り替え（ブロンズ / シルバー / ゴールド / プラチナ / M2 SPIKE / M2 EV3 / M3）
- Google Visualization API でスプレッドシートを取得
- 30秒ごとに自動更新

## 使い方

1. このリポジトリの `index.html` をブラウザで開きます。
2. タブを選択すると該当コースの順位が表示されます。

## データソース

`index.html` 内の `spreadsheetIds` に Google スプレッドシート ID が設定されています。各シートは **公開（閲覧可能）設定** が必要です。

## 注意事項

- 公開範囲と閲覧内容（個人情報の有無）を必ず確認してください。
- 外部CDN（Google Fonts / Google Charts）に依存しています。組織ポリシーがある場合はご確認ください。

## 構成

- `index.html`: メインの静的ページ
